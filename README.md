# 🏖️ The Modern

A single-page, responsive website for a fictional beach-themed contemporary restaurant — built with HTML5, CSS3, and JavaScript.

🔗 **[View Live Site](https://anoushkapanda.github.io/The_Modern_website/)**

---

## Features

- Clean, modern single-page layout
- Beach and ocean-inspired visual design
- Responsive across all screen sizes
- Smooth scrolling and interactive UI elements

## Tech Stack

- HTML5 (semantic markup)
- CSS3 (Flexbox, animations, custom properties)
- JavaScript
